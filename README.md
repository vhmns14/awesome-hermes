# Awesome Hermes [![Awesome](https://awesome.re/badge.svg)](https://awesome.re) [![Track Releases](https://github.com/vhmns14/awesome-hermes/actions/workflows/track-releases.yml/badge.svg)](https://github.com/vhmns14/awesome-hermes/actions/workflows/track-releases.yml)

> A curated list of awesome resources, models, agents, skills, tools, and community projects around **Hermes** by [Nous Research](https://nousresearch.com) — the open-source self-improving AI agent and the Hermes model family.

Hermes is two things: the **Hermes model family** (open-weight LLMs fine-tuned by Nous Research) and **Hermes Agent** (a self-hosted, model-agnostic, self-improving autonomous agent). This list collects the best things built with and for both.

## Models

- [Hermes 3 (Llama 3.1)](https://huggingface.co/collections/NousResearch/hermes-3-66bd6c01399b14b08fe335ea) — 8B / 70B / 405B open-weight fine-tunes with function calling.
- [Hermes 4](https://openrouter.ai/nousresearch/hermes-4-405b) — hybrid reasoning model (70B & 405B) on OpenRouter.
- [DeepHermes 3 (Mistral 24B)](https://openrouter.ai/nousresearch/deephermes-3-mistral-24b-preview) — reasoning-focused Mistral fine-tune.

## Tools & Integrations

- [Nous Portal](https://portal.nousresearch.com) — unified model gateway (OpenAI-compatible).
