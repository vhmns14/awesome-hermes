# Contributing to Awesome Hermes

Thanks for helping make this list awesome! Please follow these simple rules.

## Adding an entry

1. Fork the repo and create a branch.
2. Add your link under the most relevant section in `README.md`.
3. Use this format: `- [Name](https://link) — one-line description.`
4. Keep descriptions short, factual, and free of marketing fluff.
5. Make sure the link works and points to something Hermes-related.
6. Open a pull request with a clear title.

## Guidelines

- Entries must relate to **Hermes** (models or Hermes Agent) by Nous Research.
- Prefer official, well-maintained, or widely-used resources.
- No affiliate links, spam, or paid promotions.
- Dead links will be removed by the release-tracking workflow or maintainers.

## Code of Conduct

Be respectful. We follow the [Contributor Covenant](https://www.contributor-covenant.org).
